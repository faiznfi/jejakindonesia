### ğŸ—ºï¸ Tentang Proyek

**Jejak Indonesia** adalah aplikasi web berbasis peta interaktif yang memungkinkan pengguna menandai kota dan provinsi yang telah mereka kunjungi di Indonesia.

Dibuat dengan pendekatan **Mobile-First**, aplikasi ini memiliki tampilan yang bersih, modern, dan "Instagrammable", sangat cocok untuk dibagikan ke media sosial (Instagram Story). Data perjalanan disimpan secara lokal di browser pengguna, sehingga privasi tetap terjaga.

### âœ¨ Fitur Utama

* **Peta Interaktif:** Peta Indonesia lengkap (Kota & Provinsi) menggunakan teknologi Leaflet.js.
* **Deteksi Lokasi Otomatis:** Fitur cerdas yang mendeteksi lokasi pengguna saat ini dan otomatis menandai wilayah tersebut dengan notifikasi *Toast* yang elegan.
* **Smart Dashboard:**
    * **Desktop:** Tampilan *Floating Glassmorphism*.
    * **Mobile:** Tampilan *Bottom Sheet* modern (mirip Google Maps/Gojek) yang responsif.
* **Pelacakan Progress:** Statistik persentase kunjungan per pulau (Jawa, Sumatera, Kalimantan, dll).
* **Estetik & Responsif:** Desain UI minimalis, ikon SVG kustom, dan animasi halus.
* **Privasi:** Semua data tersimpan di `LocalStorage` browser pengguna (tidak ada database server).

## ğŸ‡ºğŸ‡¸ English Description

### ğŸ—ºï¸ About The Project

**Jejak Indonesia** is an interactive web-based map application that allows users to mark the cities and provinces they have visited across the Indonesian archipelago.

Built with a **Mobile-First** approach, this application features a clean, modern, and "Instagrammable" design, making it perfect for sharing on social media. Travel data is stored locally in the user's browser to ensure privacy.

### âœ¨ Key Features

* **Interactive Map:** Complete map of Indonesia (Cities & Provinces) powered by Leaflet.js.
* **Auto Location Detect:** A smart feature that detects the user's current GPS location and automatically marks the region with an elegant *Toast* notification.
* **Smart Dashboard:**
    * **Desktop:** Floating Glassmorphism view.
    * **Mobile:** Modern *Bottom Sheet* view (similar to Google Maps/Gojek).
* **Progress Tracking:** Visit percentage statistics categorized by island (Java, Sumatra, Kalimantan, etc.).
* **Aesthetic & Responsive:** Minimalist UI design, custom SVG icons, and smooth animations.
* **Privacy First:** All data is stored in the user's browser `LocalStorage` (serverless).

### ğŸš€ Getting Started

Want to run this project on your local machine? Follow these steps.

1.  **Clone this repo**
    ```sh
    git clone [https://github.com/faiznfi/jejakindonesia.git](https://github.com/faiznfi/jejakindonesia.git)
    ```
2.  **Open the folder**
    Navigate to the project directory.
3.  **Run (IMPORTANT)**
    Since this app loads external JSON files, you **must** run it using a local server to avoid CORS errors.
    * If using VS Code: Install the **Live Server** extension, open `index.html`, right-click, and select **"Open with Live Server"**.
