## ğŸ—ºï¸ About The Project

**Jejak Indonesia** adalah aplikasi web berbasis peta interaktif yang memungkinkan pengguna menandai kota dan provinsi yang telah mereka kunjungi di Indonesia. 

Dibuat dengan pendekatan **Mobile-First**, aplikasi ini memiliki tampilan yang bersih, modern, dan "Instagrammable", sangat cocok untuk dibagikan ke media sosial. Data perjalanan disimpan secara lokal di browser pengguna.

## âœ¨ Features

* **Interactive Map:** Peta Indonesia lengkap (Kota & Provinsi) menggunakan Leaflet.js.
* **Auto Location Detect:** Fitur cerdas mendeteksi lokasi pengguna saat ini dan otomatis menandai wilayah tersebut dengan notifikasi *Toast* yang elegan.
* **Smart Dashboard:**
    * **Desktop:** Tampilan Floating Glassmorphism.
    * **Mobile:** Tampilan *Bottom Sheet* modern (seperti Google Maps/Gojek).
* **Progress Tracking:** Statistik persentase kunjungan per pulau (Jawa, Sumatera, Kalimantan, dll).
* **Estetik & Responsif:** Desain UI minimalis, ikon SVG kustom, dan animasi halus.
* **Privacy First:** Semua data tersimpan di `LocalStorage` browser pengguna (tidak ada database server).

## ğŸš€ Getting Started

Ingin menjalankan proyek ini di komputermu? Ikuti langkah mudah ini.

### Prerequisites

* Browser Modern (Chrome, Edge, Safari).
* Extension **Live Server** (jika menggunakan VS Code) untuk menghindari isu CORS pada file JSON.

### Installation

1.  **Clone repo ini**
    ```sh
    git clone [https://github.com/faiznfi/jejakindonesia.git](https://github.com/faiznfi/jejakindonesia.git)
    ```
2.  **Buka folder**
    Masuk ke folder proyek.
3.  **Jalankan**
    Buka `index.html` menggunakan Live Server.
